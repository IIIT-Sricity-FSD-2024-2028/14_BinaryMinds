# Figma UI Designs – Digital Application, Approval & Licensing System
##Figma link https://www.figma.com/proto/Iiqre44YvcYXJNjQL2TrvM/ffsd?node-id=0-1&t=uhd9T6GmiGKecBRe-1

This folder contains the Figma UI designs created for the **Digital Application, Approval & Licensing System** developed as part of the **Full Stack Software Development (FFSD) project**.

The UI designs are structured based on the three main system actors.

---

## Actors and UI Modules

### 1. Applicant UI
Interfaces used by applicants to interact with the system.

Screens include:
- Landing Page
- Login / Signup
- Application Form
- Payment Page
- License Status Tracking

---

### 2. Field Officer UI
Used by field officers to perform inspections and verify documents.

Screens include:
- Officer Dashboard
- Inspection Report Submission
- Document Verification

---

### 3. Department Officer UI
Used by department officers to approve applications and issue licenses.

Screens include:
- Officer Dashboard
- Application Review
- Compliance Monitoring
- Final License Authorization

---

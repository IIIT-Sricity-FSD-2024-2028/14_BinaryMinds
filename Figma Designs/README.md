This folder contains the complete UI/UX design prototypes for the **Digital Application, Approval & Licensing Management System**.  
The designs were created using **Figma** to visualize the layout, user interactions, and workflow of the licensing platform before development.

## Purpose of the Designs
The Figma designs help demonstrate:
- The interface layout of the system
- Navigation between different modules
- Interaction flow for applicants and officers
- The complete workflow of the license application process

## Main Design Screens Included

### 1. Login Page
Allows users to log into the system.  
User roles include:
- Applicant
- Field Officer
- Department Officer

### 2. Applicant Dashboard
Features available to the applicant:
- Apply for a new trading license
- Upload required documents
- Track application status
- View issued license
- Request license renewal

### 3. License Application Form
Applicants can submit details required for a trading license such as:
- Personal information
- Business details
- Shop address
- Trade category

### 4. Document Upload Page
Applicants upload the required documents:
- Aadhaar Card
- Business Affidavit
- Passport Size Photo

### 5. Field Officer Dashboard
Field officers can:
- View assigned applications
- Verify documents
- Schedule shop inspections
- Upload inspection reports and photos

### 6. Department Officer Dashboard
Department officers can:
- Review verified applications
- Approve or reject license requests
- Digitally sign approved licenses


## Design Tool
All interface designs were created using **Figma**.
